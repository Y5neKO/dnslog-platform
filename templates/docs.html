WS ws://HOST:{{ ws_port }} WebSocket real-time updates

Connection

// 1. Connect (auto-select wss:// if behind HTTPS)
const ws = new WebSocket(
  (location.protocol === 'https:' ? 'wss:' : 'ws:') + 'HOST:{{ ws_port }}'
);

// 2. Send auth within 5 seconds
ws.send(JSON.stringify({ "code": "your_access_code" }));

// 3. Receive updates
ws.onmessage = () => {
  // Fetch latest records via GET /api/records
};

Notes

  • Must send auth message within 5s of connection, or socket is closed
  • Server sends "update" string on new record; client should fetch /api/records
  • Max 100 concurrent WebSocket connections
  • Ping/pong keepalive every 20s to detect zombie connections
  • Auto-reconnect recommended on close
